# WhatsApp-Attendance-Notification-System-using-Python-WhatsApp-Cloud-API(USING VS CODE)
Python-based attendance alert system integrated with WhatsApp Cloud API. Processes CSV-based student and attendance data, detects absentees, maps parent WhatsApp numbers, and delivers automated absence notifications in real time using Meta's WhatsApp Business platform.
## Advantages

- Automates the attendance notification process.
- Sends real-time WhatsApp notifications to parents.
- Reduces manual communication efforts by faculty and proctors.
- Improves parent awareness regarding student attendance.
- Generates personalized messages with student-specific details.
- Minimizes human errors in attendance communication.
- Uses WhatsApp Cloud API for reliable message delivery.
- Supports CSV-based student and attendance data management.
- Cost-effective compared to traditional communication methods.
- Enhances communication between educational institutions and parents.
- Easy to implement and maintain.
- Provides a scalable foundation for future communication systems.

---

## Disadvantages

- Depends on WhatsApp Cloud API availability and service uptime.
- Requires valid access tokens for authentication.
- Temporary access tokens expire and require renewal.
- Relies on accurate student and parent information in CSV files.
- Limited to users who have WhatsApp installed.
- Initial version does not include a web-based dashboard.
- CSV file management becomes difficult as data volume increases.
- No built-in role-based authentication mechanism.
- Limited message tracking and reporting features.
- Requires internet connectivity for message delivery.
- Scalability is limited without database integration.
- Incorrect attendance records may result in incorrect notifications.
